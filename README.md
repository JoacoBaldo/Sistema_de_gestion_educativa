# Sistema de Gestión Educativa
## Instalación

```bash
git clone <repo-url>
cd Sistema_gestion_educativa

python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

pip install -r requirements.txt
```

## Configuración

```bash
cp .env.example .env
```

## Ejecutar

```bash
python app.py
```

Disponible en `http://localhost:5000`
